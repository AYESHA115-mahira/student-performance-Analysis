# Student Performance Analysis

This project analyzes how different factors affect student academic performance.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Project Description
The dataset contains information about:
- Study hours
- Attendance
- Parent education
- Exam scores

The project analyzes the data and visualizes relationships between study habits and student performance.
